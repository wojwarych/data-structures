# Overview

This repository contains some Data Structures (abstract one including) implemented in Golang.
Its purpose is to learn some basic features of Go language as well as sharpen skills and knowledge in data structures in Computer Science

## Description
Each package contains some implementation of data strucutre or multiple variations of it (see for example `singlelinkedlist.go` and `doublylinkedlist.go` in `linkedlist` dir)


## Testing
For each file/package that implements a data structure there is or will be added test file with `<name-of-structure>_test.go` name in the root dir of repo.
The simplest way to run tests which check if DTs are working as expected is to run in root dir
```bash
make tests
```
